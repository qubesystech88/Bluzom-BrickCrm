# CRM Frontend (Fe)

This folder contains the React frontend for the CRM application.

## Tech Stack
- React
- TailwindCSS
- Vite
- Chart.js
- Lucide React (Icons)

## Setup
1. Navigate to this folder: `cd Fe`
2. Install dependencies: `npm install`
3. Run dev server: `npm run dev`

## Structure
- `src/components`: UI components (Sidebar, Layout, etc.)
- `src/pages`: Page components (Dashboard, Vehicles, etc.)
- `src/App.jsx`: Main routing configuration
- `src/index.css`: Tailwind directives and global styles

## Deployment
- Ready for deployment on Vercel/Netlify.
- Build command: `npm run build`
